# Kako koristiti agenta za odgovore na forumu

Kratka uputa za čovjeka. Sva "pamet" je u tri fajla, ti samo popuniš prompt i pošalješ.

## Fajlovi

- [prompt.md](prompt.md) — ovo šalješ agentu. Ima gotov predložak, ti samo popuniš sekciju Input.
- [pravila-za-pisanje-odgovora.md](pravila-za-pisanje-odgovora.md) — sva pravila forme i tona. Agent ih čita, ne treba ih ručno slati.
- [primjer-dobrog-odgovora.md](primjer-dobrog-odgovora.md) — kako izgleda dobar gotov odgovor.

## Postupak

1. Otvori prompt.md i popuni sekciju Input.
2. Pošalji cijeli prompt.md agentu.
3. Agent vrati gotov odgovor u markdownu, spreman za copy-paste na forum.

## Kako popuniti Input

- **Pitanje s foruma** — zalijepi originalni tekst korisnika, doslovno.
- **Interni linkovi** — ubaci samo farmaceut.hr i forum.farmaceut.hr linkove koje želiš u odgovoru. Agent koristi samo ove, ne traži sam.
- **Vanjski izvori** — ako imaš provjerene izvore, ubaci ih. Ako ne, ostavi prazno pa agent predloži (označi ih "(provjeriti)" da znaš provjeriti prije objave).
- **Kontekst teme** — napiši je li prvi odgovor ili nastavak. Ako je nastavak, zalijepi prethodne odgovore.
- **Rod farmaceuta** — zadano žensko, promijeni samo ako treba.
- **Dodatne napomene** — sve ostalo: na što staviti naglasak, koje doze, čega se kloniti.

## Prije objave

Brzo prelistaj odgovor i provjeri vanjske izvore označene s "(provjeriti)".
